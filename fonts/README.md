# Fonts

The ESPHome YAML uses these font files:

- `PNB.ttf`
- `PNR.ttf`

The original file names look like a bold and regular pair from a Proxima Nova-style font family. If you replace them, use your own licensed `.ttf` files with the same names, or edit the `font:` section in `portable-air-quality-monitoring-station.yaml`.

If the files are missing, ESPHome compilation will fail.
