# Fonts

The ESPHome YAML expects these font files:

- `PNB.ttf`
- `PNR.ttf`

The original file names look like a bold and regular pair from a Proxima Nova-style font family, but the exact font is not included here. Add your own licensed `.ttf` files using those names, or edit the `font:` section in `airmikol.yaml` to point at fonts you are allowed to redistribute or use.

If the files are missing, ESPHome compilation will fail.
